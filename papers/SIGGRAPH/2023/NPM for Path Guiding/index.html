Neural Parametric Mixtures for Path Guiding


论文标题：Neural Parametric Mixtures for Path Guiding

论文来源：SIGGRAPH 2023

论文作者：Honghao Dong, Guoping Wang, Sheng Li*

论文标签：Path Guiding

论文 / 项目链接

基本信息：

  • 简介：基于神经参数混合模型的路径引导。
  • 相关材料：
    • 源码：无
    • 视频：无
    • 讨论班: 2023-07-23

本文提出了一种使用神经网络编码目标分布的路径引导算法，该方法使用神经网络编码的参数模型对渲染方程里的被积函数的分布进行建模。与先前的工作（improved PPG [Müller 2019]、Variance-aware Path Guiding [Rath et al. 2020]）相比具有更好的效果与性能。
